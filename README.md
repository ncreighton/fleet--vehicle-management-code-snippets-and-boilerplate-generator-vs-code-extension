# Fleet & Vehicle Management Code Snippets and Boilerplate Generator VS Code Extension

> A comprehensive VS Code extension that accelerates fleet and vehicle management development by providing a rich library of code snippets and boilerplate generators. Designed for developers working on telematics, logistics, and transportation software, it offers pre-built templates for common tasks such as vehicle tracking, fuel management, maintenance scheduling, driver assignments, route optimization, and fleet analytics. Each snippet is optimized for modern frameworks like React, Angular, Node

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.3baff0cc`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Fleet & Vehicle Management Code Snippets and Boilerplate Generator VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `3baff0cc.enabled` | `true` | Enable/disable the extension |
| `3baff0cc.autoRun` | `false` | Auto-run on file open |
| `3baff0cc.logLevel` | `"info"` | Log verbosity |
